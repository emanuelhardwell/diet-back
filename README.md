# Nodejs-supples-backend

A full-stack e-commerce project built with Nodejs, Express, MySQL, sequelize(ORM).

## Install dependencies 📦

```
npm: npm i
yarn: yan
```

## Dev and Pro enviroment using node 📦

```
npm start
npm run dev
```

## Open your browser 🚀

[localhost:3000](http://localhost:3000)
